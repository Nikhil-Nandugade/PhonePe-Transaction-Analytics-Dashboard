# 📌 Project Overview


The **PhonePe Transaction Analytics Dashboard** is an interactive Business Intelligence project developed using **Power BI**, **Power Query**, and **Microsoft Excel** to analyze digital payment transactions across multiple PhonePe services, including **Money Transfer**, **Recharge & Bills**, **Loans**, and **Insurance**. The project transforms raw transaction data into meaningful visual insights that help monitor business performance, payment success rates, transaction trends, and customer behavior. The dashboard enables users to make data-driven decisions through interactive reports, filters, and KPIs.

---

# 🎯 Project Objectives

The main objective of the project is to collect, clean, transform, and visualize transaction data to support business decision-making. The dashboard focuses on calculating key performance indicators (KPIs) such as Total Transaction Amount, Total Transactions, Successful Transactions, Failed Transactions, and Service-wise Revenue. It also analyzes monthly transaction trends, payment failure reasons, and the performance of various services such as Loans, Insurance, Money Transfer, and Recharge & Bills. Interactive filters and navigation improve the user experience and allow business users to explore the data efficiently.
